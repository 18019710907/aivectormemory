13. 参考：服务端功能开关 → UI 映射
服务端配置类型对应 UI 元素关闭时
registration_enabledbool注册按钮 + 注册弹窗隐藏注册入口
email_verify_enabledbool验证码输入框 + 发送按钮隐藏验证码字段
promo_code_enabledbool优惠码输入框 + 验证按钮隐藏优惠码字段
invitation_code_enabledbool邀请码输入框（必填）隐藏邀请码字段
password_reset_enabledbool"忘记密码？"链接隐藏该链接
totp_enabledbool2FA 弹窗（登录响应触发）登录不返回 requires_2fa
site_namestring弹窗标题 / 品牌名默认："AVM Router"
使用的 shadcn/ui 组件：
Dialog（登录/注册/2FA/忘记密码）、Sheet side=right（用户资料）、Tabs（Memory 子视图）、
Table + DataTable（记忆/问题列表）、Form + FormField + Zod（认证表单）、
Select（项目选择器/筛选器）、Button 各变体、Badge、AlertDialog（危险操作确认）、
Sonner toast（成功/失败反馈）、Skeleton（加载状态）、Tooltip（图标按钮提示）
