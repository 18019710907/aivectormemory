# AIVectorMemory 打包说明

## 一、从上游同步最新代码

上游仓库：`https://github.com/Edlineas/aivectormemory`

```bash
git fetch upstream
git merge upstream/main
```

有冲突则解决后再：

```bash
git push origin main
```

---

## 二、打 macOS x86 包

**一键脚本（推荐）：**

```bash
./build-macos-x64.sh
```

产物在项目根目录：`AIVectorMemory-<版本>-macos-x64.zip`。需已安装 [Wails](https://wails.io)（见下方「构建桌面端」）。

---

### 手动步骤

### 1. 构建前端

```bash
cd desktop/frontend
npx vue-tsc --noEmit && npx vite build
```

### 2. 构建桌面端 (darwin/amd64)

需要先安装 Wails（未装过时执行一次）：

```bash
GOBIN=$HOME/go/bin go install github.com/wailsapp/wails/v2/cmd/wails@latest
```

然后构建：

```bash
cd desktop
CGO_ENABLED=1 GOOS=darwin GOARCH=amd64 $HOME/go/bin/wails build
```

### 3. 下载 sqlite-vec (macos x86_64)

```bash
cd 项目根目录
curl -fSL -o vec_archive.tar.gz "https://github.com/asg017/sqlite-vec/releases/download/v0.1.6/sqlite-vec-0.1.6-loadable-macos-x86_64.tar.gz"
mkdir -p vec_extract && tar xzf vec_archive.tar.gz -C vec_extract
find vec_extract -name "vec0.dylib" -exec cp {} . \;
```

### 4. 打包成 zip

版本号会自动从 `pyproject.toml` 的 `version` 读取：

```bash
VERSION=$(grep '^version' pyproject.toml | sed 's/.*= *"\(.*\)"/\1/')
PKG_NAME="AIVectorMemory-${VERSION}-macos-x64"
mkdir -p "$PKG_NAME"
cp -R desktop/build/bin/AIVectorMemory.app "$PKG_NAME/"
cp vec0.dylib "$PKG_NAME/"
cp scripts/install.sh README.md "$PKG_NAME/"
zip -r "${PKG_NAME}.zip" "$PKG_NAME"
```

产物：`AIVectorMemory-<版本>-macos-x64.zip`，解压后运行 `./install.sh` 安装。
